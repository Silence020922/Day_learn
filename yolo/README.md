## Operating environment

- python 3.9
- pytorch + cuda 12.6

# Run
1. pip

    ```
    $ python -m pip install --upgrade pip
    ```

2. pytorch + cuda

- windows: https://zhuanlan.zhihu.com/p/305854081    
- linux:  install python-pytorch-cuda

3. install requirements packages

    ```
    $ pip install -r requirements.txt
    ```

4. run

    ```
    python main.py
    ```
